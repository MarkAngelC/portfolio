HTML Cheat Sheet

Your quick reference for common HTML tags and their attributes.

HTML Elements and Descriptions
Tag Description
<html> Defines the root of an HTML document.
<head> Contains metadata and links to resources (like CSS or JavaScript).
<body> Defines the main content of the HTML document.
<div> Defines a section or division in an HTML document.
<a> Defines a hyperlink, used for linking to other pages or resources.
<p> Defines a paragraph of text.
<img> Defines an image in an HTML document. Attributes include src and alt.
<h1> to <h6> Defines HTML headings, with <h1> being the largest and <h6> the smallest.
<ul> Defines an unordered list.
<ol> Defines an ordered list.
<li> Defines a list item within an unordered or ordered list.
<table> Defines a table.
<tr> Defines a row within a table.
<td> Defines a cell within a table row. Use <th> for headers.
<form> Defines an HTML form for user input.
<input> Defines an input control in a form (like text fields, checkboxes, etc.). Attributes include type and placeholder.
<button> Defines a clickable button.
<span> Defines an inline container for text or elements.
<iframe> Defines an inline frame for embedding another HTML document.
<strong> Defines text with strong importance, typically displayed as bold.
<em> Defines emphasized text, typically displayed as italic.
